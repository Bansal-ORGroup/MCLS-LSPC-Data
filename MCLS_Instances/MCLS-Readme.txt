-MCLS refers to Multi-capacitated lot-sizing problem studied by Kulkarni and Bansal (2019). 

-For details of the problem and instance generation procedure, refer to the aforementioned article

-Instances generated are for MCLS with n=2 (2 machines) and MCLS with n=3 (3 machines)

-There are two subfolders - 2 machines and 3 machines

-Each subfolder has the corresponding instances for different time periods and different values of capacities

-each subfolder also has readme files providing more information about the datasets.
